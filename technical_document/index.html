HTML

HTML stands for HyperText Markup Language.

It is used to create the structure of web pages.

HTML uses tags like <p>, <div>, and <h1>.

Each tag serves a specific purpose in document structure.

HTML5 introduced new semantic elements.

  • <header>
  • <nav>
  • <article>
  • <section>
  • <footer>
CSS

CSS stands for Cascading Style Sheets.

It controls the presentation of web documents.

CSS allows for separation of content and design.

Selectors are used to apply styles to HTML elements.

It supports responsive design through media queries.

body {
  font-family: Arial, sans-serif;
}
JavaScript

JavaScript is a programming language for the web.

It allows for dynamic content on web pages.

You can manipulate the DOM using JavaScript.

JavaScript can be used on the server-side via Node.js.

Modern JavaScript supports modules and classes.

function greet(name) {
  return `Hello, ${name}!`;
}
APIs

APIs allow different software systems to communicate.

They provide a set of rules for accessing resources.

REST and GraphQL are popular API paradigms.

APIs are used for fetching and sending data.

JavaScript can interact with APIs using fetch or axios.

fetch('https://api.example.com/data')
  .then(response => response.json())
  .then(data => console.log(data));
Responsive Design

Responsive design ensures web pages look good on all devices.

It uses flexible layouts and images.

Media queries adjust styles for different screen sizes.

Responsive design improves usability.

CSS frameworks like Bootstrap aid in responsive design.

@media (max-width: 600px) {
  #navbar {
    display: none;
  }
}
